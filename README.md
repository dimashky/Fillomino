# Fillomino
Solving Fillomino Puzzle game with Advanced search algorithms like CSP and A*.
## Algorithms
* DFS and BFS
## Heuristics
* Minimum Remaining Values (MRV) heuristic
## Note
This solution is suited to board with MAX width or height is 10 for NOW.
